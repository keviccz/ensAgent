# Supplementary Material

`supplementary_material.pdf` contains the detailed methodology, experimental settings, prompt templates, supplementary tables, and additional figures accompanying the anonymous manuscript. This is the recommended browser-preview version and has been prepared as a browser-safe, searchable PDF to avoid rendering artifacts in web-based PDF viewers.

`supplementary_material_vector_only_for_download.pdf` preserves the original vector quality and selectable text. If its browser preview shows visual artifacts, please download and open it locally with a standard PDF reader.

Both files are intentionally author-free and are provided only as supplementary review materials while the manuscript remains under double-blind review.
